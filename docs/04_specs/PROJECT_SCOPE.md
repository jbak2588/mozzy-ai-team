# PROJECT_SCOPE.md

## Project Name

Humantric Net Indonesia Mozzy Agent Operating Model

## Purpose

`auto-company`의 구조를 평가해
Humantric Net Indonesia의 Mozzy 개발에 맞는
통제형 14-agent 운영체계를 설계한다.

## Primary Goals

1. `auto-company`의 핵심 구조를 정확히 이해한다.
2. 14개 agent를 Mozzy 제품 현실에 맞게 재정의한다.
3. 완전자율 대신 승인 기반 운영원칙으로 변환한다.
4. 구현 전 검토 가능한 계획 패키지를 만든다.

## Non-Goals

1. 무승인 자율 실행 도입
2. `bypassPermissions` 같은 권한 우회 적용
3. 앱 코드/인프라 즉시 수정
4. 프로덕션 배포 또는 운영 데이터 변경

## Success Criteria

- 14-agent 구성이 역할 단위로 명확히 설명된다.
- Mozzy에 그대로 가져올 요소와 버릴 요소가 구분된다.
- Humantric용 운영모델이 승인 기반으로 재설계된다.
- 다음 단계 구현 승인 판단이 가능해진다.
